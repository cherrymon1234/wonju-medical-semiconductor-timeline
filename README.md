# 원주 산업 발전 소개 웹페이지 (Wonju Industry Development)

원주가 **의료기기 산업**을 기반으로 성장한 배경과, 이를 뒷받침하기 위해 **반도체 소부장 산업**을 육성하게 된 과정, 그리고 두 산업이 **AI·첨단 기술과 융합**하여 미래 산업으로 발전하는 흐름을 소개하는 반응형 웹페이지입니다.

## 페이지 구성

1. **Hero** — 웹페이지 주제 소개
2. **Why Wonju** — 원주가 산업 발전에 적합했던 배경 (대학·병원·산업단지·수도권 접근성)
3. **의료기기 산업의 성장** — 1998년부터 현재까지의 성장 타임라인
4. **의료기기의 변화** — 기계 중심에서 센서·데이터·AI·반도체 기반으로의 전환
5. **반도체 소부장 산업의 성장** — 소재(Material)·부품(Component)·장비(Equipment) 및 육성 배경
6. **의료와 반도체의 융합** — 의료기기 → 센서 → 반도체 → AI 분석 → 정밀 의료로 이어지는 흐름
7. **원주의 미래 산업** — 의료기기·AI·반도체·데이터 기술 기반의 미래 전망
8. **프로젝트 소개** — 웹페이지 제작 과정 및 활용 기술

## 사용 기술

- HTML5
- CSS3 (Flexbox, Grid, CSS 변수, 반응형 디자인)
- JavaScript (Intersection Observer 기반 스크롤 애니메이션)

## 실행 방법

별도 빌드 과정 없이 `index.html`을 브라우저에서 바로 열면 확인할 수 있습니다.

## 디자인 방향

- 정부·지자체 미래 산업 소개 홈페이지 스타일
- 화이트 · 네이비 · 틸(청록) 중심 색상
- 카드 UI, 타임라인, 데이터 시각화 요소 활용
- 모바일/PC 반응형, 자연스러운 스크롤 애니메이션 적용
